# Análise de Importações – Supply Chain

Este projeto simula um cenário realista de importações internacionais e realiza uma análise de indicadores logísticos utilizando Python. É voltado para profissionais da área de Supply Chain que estão migrando para a área de Data Analytics.

## Objetivos
- Calcular o tempo total de entrega (do pedido à chegada)
- Identificar atrasos por fornecedor
- Analisar o custo médio dos embarques
- Gerar visualizações de performance

## Ferramentas utilizadas
- Python 3.x
- pandas
- matplotlib
- Jupyter Notebook ou VS Code

## Arquivos
- `importacoes.csv`: Dados fictícios de importações
- `analise_importacoes.py`: Script com as análises e gráficos
- `README.md`: Apresentação do projeto
